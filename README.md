## Prerequisites
- [EDOPro](https://projectignis.github.io/download.html) (of course)
- [YGOFabrica](https://github.com/piface314/ygo-fabrica/wiki/Install).

## Installation
In Terminal/Command Prompt/PowerShell, navigate to the MechanicalHeavenEDOPro folder and run the following:
`ygofab sync`

And if you open EDOPro, the cards will be available in the Deck Builder!
